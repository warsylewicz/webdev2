# Week 2 Assignment

## Overview

In this assignment, you will create a new Next.js project and create a React component that displays your name, course section, and a link to your GitHub repository. You will also create a new page that displays the same information.

The Next.js project that you create here will be used for all of the assignments in this course. You will add new pages to the project for each assignment.

## Part 1: Create a New Next.js Project

Follow the [Guide: Creating a New Next.js Project](/guides-and-references/creating-new-nextjs-project) to create a new Next.js project. The project name and GitHub repository name must be `cprg306-assignments`.

Create a React component in the project `/app` folder called `StudentInfo` that returns the following static information:

- Your name
- Your course section, e.g. `CPRG 306 A`
- A link to your GitHub repository

The instructor will be able to find your assignment in your GitHub repository. Make sure you have committed and pushed your changes to GitHub before the assignment deadline.
